if 的作用
------

根据表达式的某个条件或值结果来执行一组 sql 语句

Mysql 中怎么才算表达式？
---------------

*   结合文字，变量，运算符，甚至函数来组合
*   表达式可以返回 true、false、null

if 的语法格式
--------

```
IF(expr,v1,v2)
```

#### 语法格式说明

*   expr：表达式，返回 true、false、null
*   v1：当expr = true时返回 v1
*   v2：当expr = false、null 时返回v2

*   查询员工的name、在职状态

