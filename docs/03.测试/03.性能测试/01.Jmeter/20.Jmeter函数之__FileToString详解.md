作用
--

读取整个文件

语法格式
----

```
${__FileToString(C:\user\yuyq\Desktop\test1.dat,,)}
```

参数讲解
----

| 字段 | 含义 | 是否必传 |
| --- | --- | --- |
| File Name |  需要读取的文件名 | yes |
| File encoding |  *   读取文件的字符集 *   不传则用系统默认字符集 | no |
| Variable name | 返回值引用名称 | no |
