作用
--

返回各种格式的当前时间

语法格式
----

```
${__time(yyyy-MM-dd, Var)}
```

参数讲解
----

| 字段 | 含义 | 是否必传 |
| --- | --- | --- |
| Format | 时间格式，默认返回当前时间的毫秒 | no |
| Variable Name | 引用变量名 | no |

#### 常见格式的别名

*   YMD = yyyyMMdd
*   HMS = HHmmss
*   YMDHMS = yyyyMMdd-HHmmss

小栗子
---

```
${__time(YMD,)} # 20200715
```

```
${__time(YMDHMS,)} # 20200715-103559
```

```
${__time(yyyy-MM-dd HH:mm:ss,)} # 2020-07-15 10:55:27
```
