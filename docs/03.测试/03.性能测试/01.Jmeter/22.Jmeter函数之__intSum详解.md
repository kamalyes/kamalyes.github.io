作用
--

计算**两个或多个**整数值的和

语法格式
----

```
${__intSum(1,2,3)}
```

参数讲解
----

| 字段 | 含义 | 是否必传 |
| --- | --- | --- |
| First number | 第一个要添加的整数 | yes |
| Second number |  第二个要添加的整数 | yes |
| nth number | 后续继续要添加的整数 | no |
| Variable name | 引用返回值的变量名 | no |

小栗子
---

### 两位求和

```
${__intSum(1,2,)}
```

### 三位求和

```
${__intSum(1,2,3)}
```

因为**第三个值是纯数字**，所以不会被识别为变量名，只有**非纯数字**才会被识别哦 

### 五位求和带变量名

```
${__intSum(1,2,3,4,${__Random(100,999)},sum)}
```

最后是 sum，是非纯数字，所以是变量名
