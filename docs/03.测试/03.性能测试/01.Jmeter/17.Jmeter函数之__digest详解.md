作用
--

在特定的哈希算法中返回加密的值，并带有可选的 salt ，大写字母和变量名

语法格式
----

```
${__digest(MD5,123456,,,)}
```

参数讲解
----

| 字段 | 含义 | 是否必传 |
| --- | --- | --- |
| Algorithm |  用于加密的算法 | yes 
| String to encode |  将被加密的字符串 | yes 
| Salt to add | 盐值 | no 
| Upper Case value |  *   结果默认为小写（False） *   True：结果为大写 | no 
| Variable name |  引用返回值的变量名 | no 

#### 常见 Algorithm

*   MD2
*   MD5
*   SHA-1
*   SHA-224
*   SHA-256
*   SHA-384
*   SHA-512

MD5 加密的栗子
---------

#### 通过 __MD5 函数

```
${__digest(MD5,123456,,,)}
```

#### 通过 __digest 函数

```
${__MD5(123456,)}
```
