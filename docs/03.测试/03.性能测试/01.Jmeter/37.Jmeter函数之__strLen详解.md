作用
--

读取给定字符串的长度

语法格式
----

```
${__strLen(123456,)}
```

参数讲解
----

| 字段 | 含义 | 是否必传 |
| --- | --- | --- |
| String | 需要读取的字符串 | yes |
| Variable Name | 引用变量名 | no |


小栗子
---

```
${__strLen(123456,)} # 最终返回 6
${__strLen(${START.MS},)} # 返回毫秒时间戳的长度，13
```
