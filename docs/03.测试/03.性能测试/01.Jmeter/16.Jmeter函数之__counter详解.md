作用
--

获取环境变量的值

语法格式
----

```java
${__env(JAVA_HOME,var,default)}
```

参数讲解
----

| 字段 | 含义 | 是否必传 |
| --- | --- | --- |
| Enviroment Name | 环境变量名称 | yes
| Variable Name | 引用返回值的变量名 | no |
| Default value | 默认值 | no |
