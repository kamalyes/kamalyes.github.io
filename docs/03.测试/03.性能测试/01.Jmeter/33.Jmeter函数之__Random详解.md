作用
--

产生一个随机数

语法格式
----

```
${__Random(0,10, Var)}
```

参数讲解
----

| 字段 | 含义 | 是否必传 |
| --- | --- | --- |
| Minimum value | 最小值 | yes |
| Maximum value | 最大值 | yes |
| Variable Name | 可引用此函数生成随机数的变量名 | no |

小栗子
---

```
${__Random(0,10)}
```

随机生成 0 - 10 的数字

```
${__Random(0,10, num)}
```

随机生成 0 - 10 的数字，可通过 ${num} 引用此数字
