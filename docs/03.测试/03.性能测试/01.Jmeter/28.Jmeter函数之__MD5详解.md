作用
--

*   将指定的字符串 MD5 加密并返回，加密后是 32位小写
*   Jmeter 4.0 后才提供该函数哦

语法格式
----

```
${__MD5(123,)}
```

参数讲解
----

| 字段 | 含义 | 是否必传 |
| --- | --- | --- |
| String | 需要 MD5 加密的字符串 | yes |
| Variable Name |  引用变量名 | no |
