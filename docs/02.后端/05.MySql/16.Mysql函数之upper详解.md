upper 的作用
---------

将所有字符串更改为**大写**,然后返回

upper 的语法格式
-----------

```
UPPER(str)
```

#### 小栗子

```
SELECT UPPER('Hey'); # HEY

SELECT UPPER('hello world'); # HELLO WORLD

SELECT UPPER(NULL); # NULL
```
