floor 的作用
---------

向**下**取整,返回整数

floor 的语法格式
-----------

```
FLOOR(X)
```

#### 语法格式说明

返回不**大**于X的最**大**整数值

#### 小栗子

```
SELECT FLOOR(1.23); # 1

SELECT FLOOR(-1.23); # -2
```
