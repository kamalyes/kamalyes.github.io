ceiling 的作用
-----------

向 **上** 取整,ceil 函数一样效果哦

ceiling 的语法格式
-------------

```
CEILING(X)
CEIL(X)
```

#### 语法格式说明

返回不**小**于X的最**小**整数值  

#### 小栗子

```
SELECT CEILING(1.23); # 2

SELECT CEIL(-1.23); # -1
```
