year 的作用
--------

返回指定日期的年份，在1000到9999之间

year 的语法格式
----------

```
YEAR(date)
```

#### 语法格式说明

date：可以是指定的具体日期，也可以是日期表达式

#### 小栗子

```
SELECT YEAR('1987-01-01'); # 1987

SELECT YEAR(sysdate()); # 2020

SELECT YEAR(now()); # 2020
```
