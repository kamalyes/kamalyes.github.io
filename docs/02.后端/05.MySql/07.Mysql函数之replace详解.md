replace 的作用
-----------

将某些字符串替换成新的字符串,学过Python的同学,对这个应该不陌生,字符串函数也有一个replace,作用一样哈

replace 的语法格式
-------------

```
REPLACE(s,s1,s2) 
```

#### 语法格式说明

*   s：指定字符串
*   s1：需要替换掉的字符串
*   s2：新的字符串
*   **注意：**非字符串类型也是替换的,譬如 int整数、datetime日期等

#### 小栗子

```
select replace("abcd","ab","啊啊") # 啊啊cd

select replace(12341,1,"--") # --234--

select replace("二逗啊啊啊啊","啊",111) # 二逗111111111111
```
