rand 的作用
--------

生成随机数

rand 的语法格式
----------

```
RAND()
```

#### 语法格式说明

*   随机生成 0 - 1的浮点数
*   如果要指定指定范围的随机整数的话,需要用这个公式 `FLOOR(i + RAND() * j)`

#### 小栗子

```
select rand();

# 生成 7 - 12的随机数
SELECT FLOOR(7 + (RAND() * 5));
```
