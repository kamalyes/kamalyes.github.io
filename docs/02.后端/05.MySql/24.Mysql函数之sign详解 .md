sign 的作用
--------

返回参数的符号

sign 的语法格式
----------

```
SIGN(X)
```

#### 语法格式说明

*   返回 X 的符号标志,负数 = -1,整数 = 1,零 = 0
*   **注意：**也可以传字符串,中文 = 0,负数开头 = -1,正数开头 = 1,字母开头 = 0

#### 小栗子

```
SELECT SIGN(-32); # -1

SELECT SIGN(0); # 0

SELECT SIGN(234); # 1

SELECT SIGN("-1s"); # -1

SELECT SIGN("1s1"); # 1

SELECT SIGN("中文"); # 0
```
