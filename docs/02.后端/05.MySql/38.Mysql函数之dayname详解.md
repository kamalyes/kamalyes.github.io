dayname 的作用
-----------

*   返回指定日期在本周周几的全名**（默认：英文）**
*   名称使用的语言由lc_time_names系统变量的值控制

dayname 的语法格式
-------------

```
DAYNAME(date)
```

#### 语法格式说明

date：可以是指定的具体日期，也可以是日期表达式

#### 小栗子

```
SELECT DAYNAME('1987-01-01'); # Thursday

SELECT DAYNAME(sysdate()); # Sunday

SELECT DAYNAME(now()); # Sunday
```
