instr 的作用
---------

返回第一次出现子串的位置,和 locate 一样的作用,只是函数内部参数顺序调过来了而已~哈哈哈

instr 的语法格式
-----------

```
INSTR(str,substr)
```

#### 语法格式说明

*   是不是...似曾相识....对！没错！和 LOCATE(substr,str) 一样的参数,只是参数顺序反过来了！
*   返回 substr 在 str 中第一次出现的位置

#### 小栗子

```
SELECT INSTR('foobarbar', 'bar');  # 4 

SELECT INSTR('xbar', 'foobar'); # 0
```
