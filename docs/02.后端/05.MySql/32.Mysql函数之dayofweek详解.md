dayofweek 的作用
-------------

返回日期在本周的星期几索引，**星期天 = 1，星期一 = 2，星期六 = 7**

dayofweek 的语法格式
---------------

```
DAYOFWEEK(date)
```

#### 语法格式说明

date：可以是指定的具体日期，也可以是日期表达式

#### 小栗子

```
SELECT DAYOFWEEK('2008-02-03'); # 1

SELECT DAYOFWEEK(now()); # 7

SELECT DAYOFWEEK(sysdate()); # 7
```
