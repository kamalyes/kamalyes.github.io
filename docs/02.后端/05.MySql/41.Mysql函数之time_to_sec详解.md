time_to_sec 的作用
-----------------

将指定时间转换为秒

time_to_sec 的语法格式
-------------------

```
TIME_TO_SEC(time)
```

#### 语法格式说明

*   time：传入时间，如果传入了日期部分，也不会管，只将时间部分转换成秒
*   **重点：**是指将传入的时间转换成距离当天00:00:00的秒数，**00:00:00为基数，等于 0 秒**

#### 小栗子

```
SELECT TIME_TO_SEC('1987-01-01 00:39:38'); # 2378

SELECT TIME_TO_SEC(CURRENT_TIME); # 42564

SELECT TIME_TO_SEC(CURTIME()); # 42564
```
