lower 的作用
---------

将所有字符串更改为**小写**,然后返回

lower 的语法格式
-----------

```
LOWER(str)
```

#### 小栗子

```
SELECT LOWER('QUADRATICALLY'); # quadratically

SELECT LOWER('ABc123asdBBB');  # abc123asdbbb
```
