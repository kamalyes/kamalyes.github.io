month 的作用
---------

返回指定日期的月份，1月至12月的范围为1到12

month 的语法格式
-----------

```
MONTH(date)
```

#### 语法格式说明

date：可以是指定的具体日期，也可以是日期表达式

#### 小栗子

```
SELECT MONTH('2008-02-03'); # 2

SELECT MONTH(now()); # 5

SELECT MONTH(sysdate()); # 5
```
