## 作用
从容器创建一个新的镜像

## 语法格式
```python
docker commit [OPTIONS] CONTAINER [REPOSITORY[:TAG]]
```

#### options 说明
<table>
<thead>
<tr>
<td>options</td>
<td>作用</td>
</tr>
</thead>
<tbody>
<tr>
<td>-a</td>
<td>提交镜像的作者</td>
</tr>
<tr>
<td>-c</td>
<td>使用 Dockerfile 指令来创建镜像</td>
</tr>
<tr>
<td>-m</td>
<td>提交时的说明文字</td>
</tr>
<tr>
<td>-p</td>
<td>commit 时，将容器暂停</td>
</tr>
</tbody>
</table>

## 实际栗子
```bash
docker commit jenkins1 jenkins/kamalyes # 不带参数
docker commit -a yy -m kamalyes jenkins1 jenkins/kamalyes  # 带参数
```
