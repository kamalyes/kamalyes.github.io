### ONBUILD

- ONBUILD 是一个特殊的指令，它后面跟的是其它指令，比如 RUN, COPY 等，而这些指令，在当前镜像构建时并不会被执行
- 只有当以当前镜像为基础镜像，去构建下一级镜像的时候才会被执行
- Dockerfile 中的其它指令都是为了定制当前镜像而准备的，唯有 ONBUILD 是为了帮助别人定制自己而准备的

感觉实际应用场景不会很多，也不展开讲解
