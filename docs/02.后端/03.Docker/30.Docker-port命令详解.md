## 作用
列出指定的容器的端口映射

## 语法格式
```python
docker port CONTAINER [PRIVATE_PORT[/PROTO]]
```

- PRIVATE_PORT：指定查询的端口
- PROTO：协议类型（tcp、udp）

## 实际栗子
```bash
docker port gitlab 列出容器所有端口的映射
docker port gitlab 80 # 列出容器指定端口的映射
docker port gitlab 80/tcp # 列出容器指定端口和协议的映射
```
