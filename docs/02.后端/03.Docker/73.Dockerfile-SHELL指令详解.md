### SHELL
```python
SHELL ["executable", "parameters"]
```
SHELL 指令可以指定 RUN、ENTRYPOINT、CMD 指令的 shell，Linux 中默认为&nbsp;["/bin/sh", "-c"]&nbsp;

### 小栗子
```python
SHELL ["/bin/sh", "-c"]

RUN lll ; ls

SHELL ["/bin/sh", "-cex"]

RUN lll ; ls
```

- 两个 RUN 运行同一命令
- 第二个 RUN 运行的命令会打印出每条命令并当遇到错误时退出
