## 作用
从 tar 归档文件中创建**镜像**

#### 导出 tar 归档文件

## 语法格式
```python
docker import [OPTIONS] file|URL|- [REPOSITORY[:TAG]]
```

| option |	说明 |
|----|----|
-m	| 提交时的说明文字
-c	| 将Dockerfile指令应用于创建的映像

## 实际栗子
```python
docker import test.tar kamalyes/tomcat
```
