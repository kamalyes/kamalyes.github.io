## 作用
显示容器正在运行的进程

## 语法格式
```bash
docker top CONTAINER [ps OPTIONS]
```

## 实际栗子
###
```bash
docker top gitlab # 列出所有运行的进程
docker top gitlab | grep redis # 列出指定名称的正在运行的进程
```
