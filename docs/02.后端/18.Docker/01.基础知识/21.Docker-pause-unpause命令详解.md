## 作用

- **docker pause：** 暂停一个或多个容器中的所有进程
- **docker unpause：** 恢复一个或多个容器中的所有进程

## 语法格式
```python
docker pause CONTAINER [CONTAINER...]
docker unpause CONTAINER [CONTAINER...]
```

## 实际栗子
```bash
docker pause tomcat7 # 暂停 tomcat 容器的服务
docker unpause tomcat7 # 恢复 tomcat 容器的服务
```
